Sekilas Tentang Saya

Halo! Nama saya Bahrian, dan saya adalah seorang mahasiswa ilmu komputer di Universitas Pattimura. Saya sangat tertarik dengan teknologi dan dulu saya ingin sekali memasuki industri bagian pembuatan game atau console.
